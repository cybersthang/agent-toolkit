## T2 — check_subagent_transcripts() per-transcript stall + cooldown

- **Touches:** `tools/agent_supervisor.py` (extend — new check function +
  per-transcript cooldown dict)
- **Depends on:** T1
- **Acceptance:** Covers `us2-per-subagent-stall-detect`. Function:
  `check_subagent_transcripts(workspace, manifest, config, now, last_notify_per_transcript) -> dict`.
  Logic:
  1. Call `discover_sub_agent_transcripts(...)` → list.
  2. For each transcript: compute `idle = now - mtime`. If
     `idle > stall_seconds` (use `subagent_stall_seconds` if set else
     fallback `stall_seconds`, D4) AND `now - last_notify_per_transcript.get(path, 0) > notify_cooldown` (D3 per-transcript)
     → mark stalled.
  3. Return `{stalled: [paths], last_notify_per_transcript: updated_dict}`.
  Cooldown per-transcript (D3) — không suppress nhau.
- **Verification:** `pytest tests/test_sub_agent_stall_watcher.py::TestDetect::test_per_transcript_independent` exit 0.
- **Risk:** low — pure logic, mocked I/O in tests.

- **Depends on:** T1-T4
- **Acceptance:** Covers `us5-lifecycle-integrate`. Main loop mỗi iteration:
  ```python
  while True:
      # v0.24 main path (giữ nguyên)
      action, last_main_notify = check_once(workspace, main_transcript, cfg,
                                            proc_alive=..., last_notify_ts=last_main_notify)
      # v0.26 sub-agent path (auto-activate)
  ```
